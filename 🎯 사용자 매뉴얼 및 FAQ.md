## 👥 직원용 사용 가이드

### 📱 LINE을 통한 등록 방법

#### 1단계: LINE Bot 친구 추가
1. **LINE 앱을 열고** QR 코드 스캔 또는 ID 검색
2. **57TB Bot을 친구로 추가**
3. **자동 인사말 메시지 확인**

#### 2단계: 등록 링크 클릭
1. **인사말 메시지에서 파란색 링크 클릭**
   ```
   🔗 คลิกลิงก์เพื่อลงทะเบียน:
   https://liff.line.me/2007748465-jvdwq0nG
   ```
2. **LIFF 앱이 자동으로 열림**

#### 3단계: 개인정보 입력

##### 3-1. 이름 선택
- **🔍 검색창에 이름 입력** 또는 **목록에서 직접 선택**
- **자신의 이름을 클릭**하여 선택
- **"ถัดไป" (다음) 버튼 클릭**

##### 3-2. 근무 지점 선택
- **🏙️ Asoke (สาขาอโศก)** 또는 **🌆 SaiMai (สาขาสายไหม)** 선택
- **"ถัดไป" (다음) 버튼 클릭**

##### 3-3. 직급 선택
다음 중 해당하는 직급을 선택:
- **📚 STP** - 인턴/학습생 (นักเรียนฝึกงาน)
- **📞 RECEPTION** - 리셉션 (แผนกต้อนรับ)
- **🧹 CLN** - 청소 (แผนกทำความสะอาด)
- **💻 ADMIN** - 온라인 상담 (ออนไลน์คอนซัลแทนต์)
- **✂️ HS** - 헤어 디자이너 (นักออกแบบทรงผม)

##### 3-4. 팀 선택 (해당하는 경우)

**자동으로 팀이 정해지는 경우:**
- **CLN, RECEPTION, ADMIN** → Manager Team
- **SaiMai 지점의 HS, STP** → Saimai 팀

**직접 선택해야 하는 경우:**
- **Asoke 지점의 HS, STP**는 다음 중 선택:
  - **👥 อนาคต** (ทีมอนาคต)
  - **👥 มั่งมีศรีสุข** (ทีมมั่งมีศรีสุข)
  - **👥 Extra** (ทีมเอ็กซ์ตร้า)

#### 4단계: 정보 확인 및 제출
1. **✅ 입력한 모든 정보 확인**
2. **정보가 정확하면 "ยืนยัน" (확인) 버튼 클릭**
3. **수정이 필요하면 "แก้ไข" (수정) 버튼 클릭**

#### 5단계: 등록 완료
1. **✅ "ลงทะเบียนสำเร็จ!" (등록 성공) 메시지 확인**
2. **앱이 자동으로 종료됨**
3. **이제 57TB 직원 혜택을 받을 수 있습니다!**

### 📝 입력 정보 가이드

#### 직급별 상세 설명

**STP (นักเรียนฝึกงาน)**
- 인턴 또는 교육생
- 각 지점별로 다른 팀 배정
- 교육 과정 참여 대상

**HS (นักออกแบบทรงผม)**
- 헤어 디자이너
- 지점별 팀 선택 또는 자동 배정
- 고객 서비스 직접 담당

**RECEPTION (แผนกต้อนรับ)**
- 프론트 데스크 직원
- Manager Team 자동 배정
- 고객 접수 및 안내 담당

**CLN (แผนกทำความสะอาด)**
- 청소 및 관리 직원
- Manager Team 자동 배정
- 매장 환경 관리 담당

**ADMIN (ออนไลน์คอนซัลแทนต์)**
- 온라인 상담 직원
- Manager Team 자동 배정
- 온라인 고객 상담 담당

## 👨‍💼 관리자용 가이드

### 📊 직원 등록 현황 확인

#### Notion에서 확인
1. **Notion 워크스페이스 접속**
2. **"57TB Employee Master" 데이터베이스 열기**
3. **Status 필드로 필터링**:
   - **Active**: 현재 근무 중인 직원
   - **Resigned**: 퇴사한 직원

#### 등록 완료 여부 확인
- **LINE User ID 필드**가 채워진 경우: 등록 완료
- **LINE User ID 필드**가 비어있는 경우: 미등록

### 🔧 시스템 관리

#### 새 직원 추가
1. **Notion 57TB Employee Master에 새 행 추가**
2. **필수 정보 입력**:
   - Name: 직원 이름
   - ID: A57250-XX 형식
   - Status: Active
   - Branch: Asoke 또는 SaiMai
   - Position: 해당 직급
   - Team: 해당 팀 (또는 비워둠)

#### 직원 정보 수정
1. **해당 직원 행 클릭**
2. **필요한 필드 수정**
3. **자동 저장됨**

#### 퇴사 처리
1. **해당 직원의 Status를 "Resigned"로 변경**
2. **미등록 목록에서 자동 제외됨**

### 📈 통계 및 리포트

#### 등록률 확인
```
총 직원 수: Active 상태 직원 수
등록 완료: LINE User ID가 있는 직원 수
등록률 = (등록 완료 / 총 직원 수) × 100%
```

#### 지점별 현황
- **Asoke 지점**: Branch = "Asoke"인 직원 수
- **SaiMai 지점**: Branch = "SaiMai"인 직원 수

#### 직급별 현황
- **각 Position별 직원 수**
- **팀별 직원 분포**

## ❓ 자주 묻는 질문 (FAQ)

### 직원용 FAQ

#### Q1: 앱이 열리지 않아요
**A:** 다음 사항을 확인해주세요:
- LINE 앱에서 링크를 클릭했는지 확인
- 인터넷 연결 상태 확인
- LINE 앱을 최신 버전으로 업데이트
- LINE 앱을 완전히 종료 후 다시 실행

#### Q2: 내 이름이 목록에 없어요
**A:** 다음과 같은 경우일 수 있습니다:
- 아직 직원 데이터베이스에 등록되지 않음
- 이미 등록을 완료함
- 퇴사 처리됨
→ **관리자에게 문의해주세요**

#### Q3: 실수로 잘못 선택했어요
**A:** 
- 마지막 확인 단계에서 **"แก้ไข" (수정)** 버튼 클릭
- 처음부터 다시 입력 가능
- 이미 제출한 경우 **관리자에게 수정 요청**

#### Q4: 앱이 중간에 멈춰요
**A:**
- LINE 앱을 종료하고 다시 실행
- 인터넷 연결 확인
- 몇 분 후 다시 시도
- 계속 문제가 발생하면 **관리자에게 신고**

#### Q5: 등록 후 혜택은 언제부터 받을 수 있나요?
**A:**
- 등록 즉시 LINE으로 공지사항 수신 가능
- 기타 혜택은 회사 정책에 따라 다름
- 자세한 내용은 **관리자에게 문의**

### 관리자용 FAQ

#### Q1: 직원이 등록했는데 데이터가 안 보여요
**A:** 다음을 확인해주세요:
- Notion 데이터베이스 새로고침
- LINE User ID 필드에 값이 있는지 확인
- Status가 "Active"인지 확인
- n8n 워크플로우 실행 로그 확인

#### Q2: 새 직급을 추가하고 싶어요
**A:** 다음 단계를 따라주세요:
1. Notion Position 필드에 새 옵션 추가
2. HTML 코드에 새 Position 옵션 추가
3. JavaScript 로직에 새 Position 처리 추가
4. GitHub에 코드 업로드

#### Q3: 특정 직원의 등록을 취소하고 싶어요
**A:**
1. Notion에서 해당 직원의 LINE User ID 필드 비우기
2. 필요시 다른 정보도 초기화
3. 해당 직원이 다시 등록 가능해짐

#### Q4: 앱이 전체적으로 작동하지 않아요
**A:** 다음을 순차적으로 확인:
1. Netlify 배포 상태
2. n8n 워크플로우 활성화 상태
3. Notion 데이터베이스 접근 권한
4. LINE Developers Console LIFF 설정

#### Q5: 등록 통계를 어떻게 확인하나요?
**A:**
1. Notion 데이터베이스에서 필터 및 정렬 사용
2. LINE User ID 필드가 있는/없는 직원으로 그룹핑
3. 필요시 데이터 익스포트하여 별도 분석

## 🔧 문제 해결 가이드

### 일반적인 문제 해결

#### 1. 로딩이 계속 되는 경우
**증상**: 앱이 로딩 화면에서 멈춤
**해결방법**:
1. LINE 앱 완전 종료 후 재실행
2. 기기 재시작
3. LINE 앱 업데이트
4. 몇 시간 후 다시 시도

#### 2. "네트워크 오류" 메시지
**증상**: 인터넷 연결 관련 오류
**해결방법**:
1. Wi-Fi 또는 모바일 데이터 연결 확인
2. 다른 웹사이트 접속 테스트
3. VPN 사용 시 해제 후 재시도
4. 네트워크 설정 초기화

#### 3. "앱을 시작할 수 없습니다" 오류
**증상**: LIFF 초기화 실패
**해결방법**:
1. LINE 외부 브라우저에서 열기 시도
2. LINE 앱 캐시 삭제
3. LINE 로그아웃 후 재로그인
4. 다른 기기에서 시도

#### 4. 데이터가 저장되지 않는 문제
**증상**: 등록 완료했지만 Notion에 반영 안됨
**해결방법** (관리자):
1. n8n 워크플로우 실행 로그 확인
2. Notion API 연결 상태 확인
3. 데이터베이스 권한 설정 재확인
4. 직원에게 재등록 요청

### 브라우저별 문제 해결

#### LINE 내장 브라우저
- **가장 권장되는 환경**
- 문제 발생 시 LINE 앱 재시작

#### Safari (iOS)
- LIFF SDK 호환성 최적화됨
- 개인정보 보호 설정으로 인한 제한 가능

#### Chrome (Android)
- 안정적인 동작
- 캐시 삭제로 대부분 문제 해결

### 기기별 최적화

#### iPhone
- iOS 12.0 이상 권장
- Safari 쿠키 설정 확인

#### Android
- Android 7.0 이상 권장
- Chrome 최신 버전 사용
