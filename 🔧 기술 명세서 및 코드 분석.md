```

5: Confirmation (정보 확인)
6: Success (완료)
```

### 4. 네트워크 재시도 로직
```javascript
async function fetchWithRetry(url, options, maxRetries = 3) {
    for (let i = 0; i <= maxRetries; i++) {
        try {
            const response = await fetch(url, options);
            if (response.ok) return response;
            
            if (i === maxRetries) throw new Error('Max retries reached');
            
        } catch (error) {
            if (i === maxRetries) throw error;
            
            // 지수 백오프 (1초, 2초, 4초 대기)
            await new Promise(resolve => 
                setTimeout(resolve, Math.pow(2, i) * 1000)
            );
        }
    }
}
```

```javascript
async function measureApiCall(apiCall, name) {

    try {
        const result = await apiCall();
        const endTime = performance.now();
        
        log(1, `API ${name} completed`, {
```

```javascript
function getMessage(key, lang = 'th') {
    return MESSAGES[lang][key] || MESSAGES.th[key] || key;
}
```
